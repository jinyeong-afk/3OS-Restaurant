### 3OS-Restaurant
3학년 운영체제 과제: 멀티스레드를 활용한 간단한 프로그램 개발

### 개요
멀티스레드에 대한 이해를 돕고자 멀티스레드를 활용한 간단한 프로그램을 개발하였다.
손님이 입장 스레드, 조리 스레드, 손님 퇴장 스레드, 화면 출력 스레드를 따로 두어 멀티스레드를 적용시켰다.
